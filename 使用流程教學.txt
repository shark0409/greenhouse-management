智慧溫室管理網站 使用流程教學

一、網站用途
這個網站是提供管理者在本機電腦上記錄與管理溫室作業使用。
目前資料儲存在本機資料夾，不是雲端版本。

可使用功能：
1. 每日管理日誌
2. 重要事件流
3. 行事曆排程
4. 今日待辦
5. 進度追蹤
6. 感測資料檔案匯入與保存
7. 資料匯出備份
8. 天氣特報提醒


二、啟動方式
1. 進入「智慧溫室\管理網站」資料夾。
2. 雙擊「啟動本機管理網站.bat」。
3. 等待命令視窗出現。
4. 瀏覽器開啟網站：
   http://127.0.0.1:8088/

注意：
1. 使用期間不要關閉命令視窗。
2. 若關閉命令視窗，網站的本機儲存功能會失效。


三、日常使用流程
建議每天依照下面順序操作：

1. 查看總覽
   - 確認今日待辦
   - 確認天氣特報
   - 確認近期日誌與重要事件
   - 確認進度追蹤

2. 填寫每日管理日誌
   - 進入「每日管理日誌」
   - 輸入日期、作業類型、負責人、狀態、內容
   - 按「新增日誌」

3. 匯入當日感測資料
   - 在「當日感測資料匯入」區塊
   - 選擇 Excel 或 CSV 檔案
   - 輸入備註
   - 按「儲存檔案」

4. 補充重要事件
   - 若當天有異常、警告、設備問題、資料問題
   - 在「新增重要系統日誌」中填寫
   - 按「新增系統日誌」

5. 檢查行事曆
   - 進入「行事曆」
   - 檢查今天是否有巡檢或其他安排
   - 若有新排程可直接新增

6. 更新今日待辦
   - 在總覽的「今日待辦」區塊
   - 可手動新增待辦事項
   - 可勾選完成
   - 可刪除手動新增的待辦
   - 今天的行事曆事件會自動帶入今日待辦

7. 更新進度追蹤
   - 進入「進度追蹤」
   - 新增項目或調整進度百分比
   - 可刪除不需要的項目


四、各功能簡單說明

1. 總覽
   用來快速查看：
   - 今日待辦
   - 近期管理日誌
   - 近期排程
   - 進度摘要
   - 嚴重天氣警示

2. 每日管理日誌
   用來記錄：
   - 巡檢
   - 灌溉
   - 施肥
   - 採收
   - 維護

3. 重要事件流
   用來記錄：
   - 系統異常
   - 資料異常
   - 警告事項
   - 需要追蹤的事件

4. 行事曆
   - 每天預設有巡檢
   - 可新增排程、提醒、會議
   - 可刪除自行新增的項目
   - 當天的行事曆事件會同步到今日待辦

5. 今日待辦
   - 可手動新增待辦
   - 可刪除手動待辦
   - 可勾選完成
   - 會自動帶入今天的行事曆事項

6. 進度追蹤
   - 可新增進度項目
   - 可調整百分比
   - 可刪除項目

7. 管理文件
   - 可快速跳到日誌與行事曆
   - 可匯出備份資料


五、資料儲存位置
網站主要資料儲存在：
本機資料庫\app-state.json

感測檔案與上傳資料儲存在：
本機資料庫\

依副檔名與日期分類，例如：
1. 本機資料庫\csv\
2. 本機資料庫\xlsx\
3. 本機資料庫\xls\


六、備份方式
1. 進入「每日管理日誌」
2. 在「資料管理」區塊按「匯出」
3. 下載 JSON 備份檔

建議：
1. 每週至少匯出一次
2. 重要修改後立即匯出一次
3. 同時備份整個「管理網站」資料夾會更完整


七、搬移到其他電腦的方法
若之後要搬到學校實驗室電腦，建議一起帶走以下內容：

1. 整個「管理網站」資料夾
2. 本機資料庫資料夾
3. app-state.json
4. 匯出的 JSON 備份檔

搬移後：
1. 在新電腦解壓或複製資料夾
2. 雙擊「啟動本機管理網站.bat」
3. 確認網站可正常開啟


八、常見問題

問題1：按儲存沒有反應，或出現 Failed to fetch
原因：
通常是本機伺服器沒有啟動。
處理方式：
1. 確認是否有執行「啟動本機管理網站.bat」
2. 確認命令視窗沒有被關掉
3. 重新整理瀏覽器

問題2：資料不見了
處理方式：
1. 檢查本機資料庫\app-state.json 是否還在
2. 使用先前匯出的 JSON 備份匯入

問題3：天氣沒有更新
處理方式：
1. 確認網路正常
2. 確認氣象署授權碼設定正確
3. 按側欄的「更新」


九、使用建議
1. 每日先看總覽，再填寫日誌。
2. 感測資料匯入後，再補重要事件說明。
3. 每週固定做一次資料匯出備份。
4. 若多人共用同一台電腦，操作前先確認日期與內容。
5. 若有重大異常，除了記錄在網站，也要另外通知負責老師或管理者。


十、目前版本限制
1. 目前是本機版，不是多人即時同步版。
2. 需要透過本機啟動檔執行，不能只直接打開 HTML。
3. 若換電腦使用，需要手動搬移資料夾或匯入備份。
