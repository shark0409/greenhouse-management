# 智慧溫室農場管理網站

這是一個本機管理版靜態網站，適合在固定管理電腦上填寫與保存智慧溫室管理資料。

資料會儲存在網站資料夾內的 `本機資料庫/`，不會上傳到雲端，也不會與其他手機或電腦同步。

## 網站檔案

- `index.html`
- `styles.css`
- `script.js`
- `server.py`
- `啟動本機管理網站.bat`
- `cwa_config.json`
- `cwa_config.example.json`
- `assets/`
  - `fonts/LXGWWenKai-Medium.ttf`
- `本機資料庫/`

使用或搬移時請保留整個 `管理網站` 資料夾內的內容。

## 本機使用

請執行 `啟動本機管理網站.bat`，再用瀏覽器開啟：

```text
http://127.0.0.1:8088/
```

若直接開啟 `index.html`，基本畫面可以看，但 Excel / CSV 檔案無法寫入本機資料庫資料夾。

## 每日氣象特報

總覽頁會透過本機 Python 服務讀取中央氣象署開放資料平台的天氣警特報資料，預設農場位置為：

```text
高雄市鳳山區
```

授權碼與位置設定放在 `cwa_config.json`。這個檔案只供本機使用，已加入 `.gitignore`，避免授權碼被上傳到 GitHub 或其他遠端服務。

若搬到實驗室電腦，請一起複製 `cwa_config.json`。如果需要重建設定，可複製 `cwa_config.example.json` 並改名為 `cwa_config.json`，再填入中央氣象署授權碼。

氣象特報需要網路連線。若授權碼錯誤、尚未啟用或氣象署 API 暫時無法連線，總覽頁會顯示錯誤提醒，不會影響日誌與檔案管理功能。

總覽頁的特報區會：

- 顯示目前最高等級警示與管理建議
- 每 5 分鐘自動更新一次
- 可按「更新特報」立即重新抓取最新資料

## 備份與還原

- 按「匯出」可下載目前資料的 JSON 備份。
- 按「匯入」可選擇先前匯出的 JSON 備份並還原資料。
- 建議每天或每週匯出備份一次，放到隨身碟、NAS 或雲端硬碟。
- Excel / CSV 感測資料檔案會保存在 `本機資料庫/`，不包含在 JSON 備份中。

## 搬到實驗室電腦

1. 在目前電腦按「匯出」，保存一份 JSON 備份。
2. 將整個 `管理網站` 資料夾複製到隨身碟或實驗室電腦。
3. 在實驗室電腦執行 `啟動本機管理網站.bat`。
4. 用瀏覽器開啟 `http://127.0.0.1:8088/`。
5. 如需沿用目前日誌資料，按「匯入」選擇 JSON 備份。
6. Excel / CSV 檔案若已在 `本機資料庫/` 內，會跟著資料夾一起搬移。
7. 之後固定在實驗室電腦使用，檔案會繼續保存到 `本機資料庫/`。

## 字體

網站已內建 `LXGW WenKai Medium` 字體檔，瀏覽器會從 `assets/fonts/` 載入，不需要在實驗室電腦另外安裝字體，也不需要連網下載。

## 資料儲存方式

此版本會將日誌、系統事件、行事曆、待辦與進度資料寫入 `本機資料庫/app-state.json`。

Excel / CSV 檔案使用本機 Python 服務保存到 `本機資料庫/`：

```text
本機資料庫/
  app-state.json
  metadata.json
  xlsx/
  xls/
  csv/
  other/
```

`app-state.json` 會保存每日日誌、系統日誌、行事曆與進度。檔案資料則依副檔名與日期分類，搬移整個 `管理網站` 資料夾即可保留全部內容。

如果未來需要多人共用同一份日誌、帳號登入、手機與電腦同步資料，才需要加上後端資料庫，例如 Firebase、Supabase、Google Sheets API 或自架伺服器。
